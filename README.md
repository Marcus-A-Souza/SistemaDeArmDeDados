Compartilhando um projeto em C# que demonstra como criar uma coleção de usuários e realizar operações básicas, como adicionar, remover e pesquisar usuários.
	 Este código é um exemplo simples de como aplicar conceitos de Programação Orientada a Objetos (POO) ,utilizando classes para representar entidades (no caso, usuários) e encapsular operações relacionadas a essas entidades.

	Nesse projeto define-se uma classe User que encapsula informações de usuário, como ID, nome e email. Implementa uma classe UserCollection que gerencia uma coleção de usuários e fornece métodos para adicionar, remover, pesquisar e exibir usuários.
Interage com o usuário por meio do console, permitindo a execução de operações específicas na coleção de usuários.
